Details of WIKIPEIA PARSER.py are as follows:-
Written in Python Version 3.8.2
This Program uses browser to search into WIKIPEDIA and find article related to your search.

Program will return the first Wikipedia result from google if your search is not realted to any article in Wikipedia.

Download Chromedriver for your chrome version and give the value of its path to the variable "executable_path" in the program.

Check Your Chrome Version:- 
https://www.whatismybrowser.com/detect/what-version-of-chrome-do-i-have

Download Chromedriver:-
 https://chromedriver.chromium.org/downloads

Refer if you wish to use other browser or solve errors:-
https://www.selenium.dev/documentation/en/webdriver/driver_requirements/

Import BeautifulSoup,requests,Selenium before running the script
